# MXstoreCMS 任务文件夹

5. `04-database-supabase.md`：Supabase 数据库与 RLS
6. `05-openlist-download-system.md`：OpenList 签名下载系统
7. `06-admin-cms.md`：管理员后台开发任务
8. `07-user-console.md`：用户后台开发任务
9. `08-auth-wallet-oauth.md`：钱包登录与第三方登录
10. `09-blockchain-payment.md`：区块链支付
11. `10-testing-security.md`：测试与安全
12. `11-deployment-vercel-supabase.md`：Vercel + Supabase Cloud 部署
13. `12-final-acceptance.md`：最终验收清单
