# 02 UI 设计系统：白色极简后台

## 1. 设计关键词

- 简洁
- 白色
- 纯色
- 现代 SaaS
- 管理后台
- 轻边框
- 大留白
- 低饱和

## 2. 页面结构

后台采用：

```txt
左侧菜单 + 顶部状态栏 + 内容区
```

## 3. 左侧菜单

菜单项：

- 仪表盘
- 应用管理
- 下载链接
- 域名与 Token
- 流量套餐
- 订单支付
- 用户管理
- 数据统计
- 系统设置

## 4. 顶部状态栏

包含：

- 搜索框
- Supabase Cloud 状态
- Vercel 部署状态
- 钱包余额或收款摘要
- 管理员头像

## 5. 仪表盘卡片

指标：

- 应用总数
- 今日下载
- 付费订单
- 活跃用户

## 6. 表格规范

表格列：

- 名称
- 版本
- 价格
- 下载链接数量
- 状态
- 操作

状态标签：

- 已发布：绿色
- 草稿：橙色
- 停用：灰色
- 失败：红色

## 7. 色彩

建议 Tailwind 配色：

```txt
背景：bg-white / bg-slate-50
边框：border-slate-200
主文字：text-slate-950
次文字：text-slate-500
主色：blue-600
成功：emerald-600
警告：amber-600
错误：rose-600
```

## 8. 禁止事项

- 不使用暗黑主题作为默认后台
- 不使用复杂渐变
- 不使用花哨背景
- 不使用大面积高饱和颜色
- 不把业务状态隐藏在纯图标里
