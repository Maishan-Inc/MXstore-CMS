# 10 测试与安全

## 1. 单元测试

必须覆盖：

- OpenList URL 解析
- 域名匹配
- sign 生成
- 过期时间生成
- 外部链接识别
- 用户权限判断
- 流量扣减逻辑

## 2. 集成测试

测试场景：

1. 管理员配置域名 Token。
2. 管理员创建应用。
3. 管理员添加 OpenList 下载链接。
4. 用户登录。
5. 用户下载免费应用。
6. 用户购买付费应用。
7. 用户购买流量套餐。
8. 用户下载付费应用。

## 3. 安全测试

检查：

- 前端 bundle 不包含 service role key。
- 前端 bundle 不包含 OpenList Token。
- 普通用户不能访问 `/admin`。
- 普通用户不能读取其他用户订单。
- 未购买付费应用不能下载。
- 流量不足不能下载。

## 4. 生产前检查

- RLS 已启用。
- 环境变量已配置。
- OpenList 已开启签名。
- Vercel 域名已绑定。
- Supabase Auth Redirect URL 已配置。
