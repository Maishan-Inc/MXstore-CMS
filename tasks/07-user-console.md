# 07 用户后台开发任务

## 1. 用户后台首页

路径：

```txt
/account
```

展示：

- 流量余额
- 已购应用
- 最近下载
- 最近订单

## 2. 我的应用

路径：

```txt
/account/apps
```

展示：

- 已购买应用
- 免费下载历史
- 继续下载按钮

## 3. 我的流量

路径：

```txt
/account/traffic
```

展示：

- 当前余额
- 流量流水
- 套餐购买入口

## 4. 我的订单

路径：

```txt
/account/orders
```

展示：

- 订单号
- 类型
- 金额
- 状态
- tx hash

## 5. 我的下载

路径：

```txt
/account/downloads
```

展示：

- 应用
- 下载链接名称
- 下载时间
- 消耗流量
- IP / User Agent 可选

## 6. 用户下载流程

1. 用户点击下载。
2. 未登录则跳转登录。
3. 如果应用付费且未购买，跳购买。
4. 如果流量不足，跳套餐购买。
5. 后端生成签名链接。
6. 302 跳转下载。
